02 // Architecture

Three hops. Six layers.

Every connection traverses guard, middle, and exit relays. Each hop adds a layer of post-quantum hybrid encryption. Hidden services add 3 more hops with vanguard protection.

Link Layer

Noise_IK + Kyber768 mandatory PQ hybrid handshake. XChaCha20-Poly1305 AEAD. 532-byte wire frames. No downgrade path.

Circuit Layer

Layered ChaCha20 onion encryption. X25519 + Kyber768 CKE per hop. Fixed 514-byte cells. BLAKE2b running digest authentication.

End-to-End (HS)

Hidden service rendezvous adds 3 vanguard-protected hops (6 total). PQ hybrid e2e keys. Any TCP service: SSH, HTTP, IRC, SMTP.

Descriptor Layer

Encrypted descriptors with signed revision counters. DPF-PIR private lookup. Blinded keys rotate per time period. Anti-replay protection.

03 // Pluggable Transports

Six ways to disappear.

When plain encryption isn't enough, make MOOR traffic look like something else entirely.

ShitStorm
Recommended

Chrome 146 JA4 fingerprint. Elligator2 key hiding. ECH GREASE. HTTP/2 framing. Key rotation every 65K records.

Nether
New

Minecraft 1.21.4 protocol camouflage. Real handshake, login sequence, and Plugin Channel framing. DPI sees Minecraft gameplay.

Mirage

TLS 1.3 framing with configurable SNI. Looks like a connection to any domain you choose.

Shade

Elligator2 point obfuscation with inter-arrival time modes. Wire bytes indistinguishable from random.

Scramble

ASCII HTTP prefix + ChaCha20 stream cipher. Passes casual DPI as regular web traffic.

Speakeasy

SSH protocol camouflage with real banner exchange. Looks like an SSH session to any observer.

04 // Quick Start

Running in 30 seconds.

One command to install. One command to run.

Quick Install

# Install MOOR curl -sL https://raw.githubusercontent.com/0xdeadbeefnetwork/MOOR_PQ/main/setup.sh | sudo bash # Run as SOCKS5 proxy on :9050 moor

Then configure Firefox: Settings → Network → SOCKS5 → 127.0.0.1:9050

Build from Source

git clone https://github.com/0xdeadbeefnetwork/MOOR_PQ cd MOOR_PQ make sudo make install # Run as SOCKS5 proxy moor

Requires: build-essential, libsodium-dev, zlib1g-dev

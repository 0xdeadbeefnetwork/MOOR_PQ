02 // Architecture

Three hops. Six layers.

Link Layer

Noise_IK + Kyber768 authenticated handshake. ChaCha20-Poly1305 AEAD. 532-byte wire frames (514 plaintext + MAC).

Circuit Layer

Layered ChaCha20 onion encryption. X25519 + Kyber768 CKE per hop. Fixed 514-byte cells prevent size fingerprinting.

End-to-End (HS)

Hidden service rendezvous adds 3 more hops (6 total). PQ hybrid e2e keys derived at rendezvous point.

03 // Pluggable Transports

Five ways to disappear.

When plain encryption isn't enough, make MOOR traffic look like something else entirely.

ShitStorm
Recommended

Chrome 146 JA4 fingerprint. Elligator2 key hiding. ECH GREASE. HTTP/2 framing. Key rotation every 65K records.

Mirage

TLS 1.3 framing with configurable SNI. Looks like a connection to any domain you choose.

Shade

Elligator2 point obfuscation with inter-arrival time modes. Wire bytes indistinguishable from random.

Scramble

ASCII HTTP prefix + ChaCha20 stream cipher. Passes casual DPI as regular web traffic.

Speakeasy

SSH protocol camouflage with real banner exchange. Looks like an SSH session to any observer.

04 // Quick Start

Running in 30 seconds.

One command to install. One command to run.

Quick Install

# Install MOOR curl -sL https://raw.githubusercontent.com/0xdeadbeefnetwork/MOOR_PQ/main/setup.sh | sudo bash # Run as SOCKS5 proxy on :9050 moor

Then configure Firefox: Settings → Network → SOCKS5 → 127.0.0.1:9050

Build from Source

git clone https://github.com/0xdeadbeefnetwork/MOOR_PQ cd MOOR_PQ ./configure make -j$(nproc) # Optional: run tests + install make check sudo make install

Requires: build-essential, libsodium-dev, zlib1g-dev, pkg-config
